Ejercicio 4

Un numero palindrómico se lee igual ​de izquierda a derecha y de derecha a izquierda. El palíndromo más grande obtenido de la multiplicación de dos números de dos dígitos es 9009 = 91x99.

Encontrar el palíndromo más grande obtenido de la multiplicación de dos números de tres dígitos.

Soluciones propuestas

Solución 1

def split(word):
            return [char for char in word]

num1 = 100
mayor = 1

for i in range(100,1000):
            num2 = num1
            for j in range(num1,1000):
                        num = num1*num2
                        numstr = str(num)
                        numintermedio = split(numstr)
                        paco = ''
                        for k in range(len(numintermedio)-1,-1,-1):
                                    paco = paco + numintermedio[k]
                        paco = int(paco)
                        apoyo = num - paco
                        if (apoyo == 0 and num > mayor):
                                    mayor = num
                        num2 += 1
            num1 += 1
